# 🛒 ECOM Express — E-Commerce Sales Dashboard (Power BI)

## 📊 Project Overview
An interactive Power BI dashboard built to analyze e-commerce 
sales performance for ECOM Express — covering revenue trends, 
product category performance, regional sales, and order 
cancellation insights.

## 🗓️ Project Date
January 2026

| Average Order Value | Rs 112.57K |
| Lost Revenue (Cancellations) | Rs 470.37M |
| Total Orders | 14.10K |

## 🔍 Key Insights
- **Laptops** are the top revenue category at Rs 0.35bn
- **MacBook** is the highest revenue-generating product
- **Maharashtra** leads all states in revenue
- Revenue peaks in **Q3** then sharply drops in Q4
- Cancellation rate of **29.61%** signals a major 
  business risk worth investigating
